Web制作・マーケティング支援

「きれいなだけのサイト」ではなく、
事業の目的達成に貢献する Web とマーケティングを設計します。

こんなお悩みはありませんか？

  • 今のホームページが古く、会社の魅力を十分に伝えられていない
  • 問い合わせ数が少なく、「名刺代わり」で終わってしまっている
  • SEO や広告、SNS を試してみたが、効果がよく分からない
  • そもそも何から手をつければよいか分からない
  • 制作会社とのやり取りに時間を取られ、本業に集中できない

EXPAND ができること

1. 目的とターゲットを明確にしたサイト設計

「誰に・何を伝え・どんな行動をしてほしいのか」を整理し、
目的に合ったサイト構成・導線を設計します。

2. コーポレートサイト制作

デザイン、コピー、構成を含めてトータルでサポートします。
WordPress / 静的HTML / Notion など、ご要望と運用体制に合わせて構築方法を選定します。

3. SEO・コンテンツ設計

検索ニーズの調査を行い、狙うべきキーワードとコンテンツの方向性を整理。
内部対策や構造設計も含めて、検索されやすい土台をつくります。

4. 広告・SNSを含めたマーケティング支援

必要に応じて、Google広告やMeta広告、SNS運用の方針づくりも支援します。
「とりあえず出す」のではなく、数字を見ながら小さく検証していくスタイルです。

ご支援の流れ

  1. 現状サイト・集客状況のヒアリング
  2. 目的・ターゲット・競合の整理
  3. サイト構成案・ワイヤーフレームの作成
  4. デザイン・コピー制作
  5. 実装・テスト・公開
  6. 公開後の改善提案（アクセス解析を含む）

期待できる効果

  • 自社の強みや価値が、初めての方にも分かりやすく伝わる
  • 問い合わせや資料請求など、具体的なアクションが増える
  • 営業前の「第一印象」を整えられることで、商談がスムーズになる
  • Web・デジタルまわりの方針が整理され、迷いが減る
